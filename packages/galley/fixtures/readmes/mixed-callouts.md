# mixed-callouts

Both callout syntaxes, mixed in one document, must render as styled callouts with
legible labels and AA-contrast text, not as raw blockquotes.

> [!NOTE]
> A GitHub-flavored note callout.

> [!WARNING]
> A GitHub-flavored warning, which should read as a caution color.

:::tip
An Obsidian-style tip callout, which must match the GitHub note styling contract.
:::

:::danger
An Obsidian-style danger callout with a longer body that wraps across the measure
so the callout padding and label color are both exercised on more than one line.
:::
